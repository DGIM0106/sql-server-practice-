# SQL Server Practice

> Repositorio de prácticas de SQL Server que te permitirán entender el funcionamiento básico del motor de bases de datos.

## ¿Qué aprenderás aquí?
* Creación de bases de datos
* Creación de tablas y definición de tipos de datos
* Inserción y manipulación de registros (DML)
* Diseño y consulta de tablas relacionales (PK, FK e INNER/LEFT JOINs)

## Tecnologías
* SQL Server (Transact-SQL)
* Azure Data Studio / SQL Server Management Studio (SSMS)
